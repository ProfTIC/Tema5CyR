3.2. Menú insertar

Tenemos tres grupos de elementos que podemos insertar en la página web:

  • Elementos externos
  • Bloques de contenido
  • Botones y elementos interactivos

A continuación vamos a explicar cada uno de ellos.

Elementos externos

Podemos insertar varios tipos de elementos externos en cualquier página:

  • Si seleccionamos cuadro de texto, se incluirá una caja de texto en la posición en la que tengamos el cursor
  • En el caso de elegir la opción imágenes, se añadirá una foto. En este caso, podremos elegir entre subir un archivo nuevo o bien buscar el enlace a través de Google Drive o Google Imágenes
  • La tercera opción, insertar, permite incluir código HTML directamente en la página.
  • Si se selecciona la última opción, Drive, podremos subir un archivo de cualquier tipo que tengamos almacenado en Drive
