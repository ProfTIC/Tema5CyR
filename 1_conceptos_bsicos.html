1. Conceptos básicos

 
 Una página web es un documento alojado en un sitio web que puede ser visitado por un usuario mediante un navegador web y que contiene información de todo tipo con diferentes formatos.

¿Qué puede haber en una página web?

  • Texto
  • Imágenes
  • Audios
  • Vídeos
  • Enlaces
  • ...

¿Cómo se crea una página web?

Una página web está escrita en el lenguaje HTML (Hypertext Markup Language). Para crear una página web podemos usar cualquier editor de texto, guardando el archivo con la extensión html. Por ejemplo, copia el siguiente código en un archivo de texto y guárdalo como index.html:

<html>
<head>
    <title>Mi primera página web</title>
</head>
<body>
    He creado mi primera página web con HTML
</body>

</html>
