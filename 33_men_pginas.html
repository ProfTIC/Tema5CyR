3.3. Menú páginas

Este menú permite ver las páginas que tenemos en nuestro sitio web. Nos aparecerá algo similar a la siguiente imagen:

Podemos modificar cada página pulsando sobre los tres puntos que aparecen junto a su nombre. Se mostrará una ventana como la siguiente:

Como se puede observar, tenemos varias opciones:

  • Establecer como página principal
    • Si marcamos esta opción será la primera página que se muestre cuando se visualice el sitio web
  • Duplicar una página
    • Se creará otra página igual que la que tenemos.
  • Propiedades
    • Se mostrarán las propiedades de la página. En concreto, podremos cambiarle el nombre y la ruta
  • Añadir una subpágina
    • Permite añadir una página nueva
  • Ocultar de la navegación
    • La página no se verá en el sitio, pero seguirá estando y podremos modificar su contenido. Esta opción es muy útil cuando queremos cambiar alguna página pero no queremos que sea visible mientras se realizan los cambios
  • Eliminar
    • Borra la página

También podemos crear una página nueva pulsando sobre el botón + en la parte inferior del menú:

En este caso, se permitirá crear una nueva página, una nueva sección de menú o un enlace
